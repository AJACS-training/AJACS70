----

## 概要

本講習は、だれでも自由に使うことができる公共DBやウェブツールを活用して、オミックス解析の基本であるゲノム配列のDBを簡単に調べるための方法と基礎知識について学びます。
とくに、それらを活用した配列解析の基礎に関して実例を交えながら詳しく説明、実習します。

----
